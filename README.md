# recherche-d-un-motif-dans-un-texte
rechercher un motif dans un texte (ctrl + F)
1) Ecrire une fonction 𝒍𝒆𝒄𝒕𝒖𝒓𝒆_𝒇𝒊𝒄𝒉𝒊𝒆𝒓(𝒏𝒐𝒎) qui retourne une chaine de caractère
correspondant au texte du fichier dont le nom est en paramètre (str).
2) Ecrire une fonction 𝒓𝒆𝒄𝒉𝒆𝒓𝒄𝒉𝒆(𝒎𝒐𝒕𝒊𝒇, 𝒕𝒆𝒙𝒕𝒆) qui recherche la chaine 𝑚𝑜𝑡𝑖𝑓 donnée en
paramètre dans la chaine 𝑡𝑒𝑥𝑡𝑒. Cette fonction retourne une liste des positions du motif dans
le texte (indice du premier caractère du motif dans le texte).
3) Ecrire une fonction 𝒕𝒆𝒎𝒑𝒔_𝒆𝒙𝒆𝒄𝒖𝒕𝒊𝒐𝒏(𝒇𝒐𝒏𝒄𝒕𝒊𝒐𝒏,𝒎𝒐𝒕𝒊𝒇, 𝒕𝒆𝒙𝒕𝒆) qui retourne le temps
d’exécution en secondes de la fonction en paramètre pour rechercher le motif dans le texte.
